## Siv3D on Docker

![screenshot](screenshot.png)

### Usage

```bash
$ pwd
/home/user/OpenSiv3D/Linux
$ mkdir App/build
$ cd App/build
$ cmake -GNinja -DCMAKE_BUILD_TYPE=RelWithDebInfo ..
$ cd ..
$ cmake --build build
```
